Scene 8 - Mark Setting Mode

Controls:

• Mouse Drag: Rotate view

• Mouse Wheel: Zoom

• Right Click + Drag: Pan


Set Character Marks:

Use console to move characters:

scene.characters.marcus.getModel().position.set(x, 0.1, z)

Log All Positions:

Object.entries(scene.characters).forEach(([id, char]) => { const pos = char.getModel().position; console.log(`${id}: { x: ${pos.x.toFixed(1)}, y: 0.1, z: ${pos.z.toFixed(1)} }`); });
